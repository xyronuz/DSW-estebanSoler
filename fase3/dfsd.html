Fundamentos de CSS

Propósito del sitio:

Las hojas de estilo en cascada (CSS, por sus siglas en inglés) son un lenguaje de diseño utilizado para definir la apariencia y la presentación de elementos en una página web. CSS permite a los desarrolladores web separar el contenido de una página (que se escribe en lenguajes como HTML o XML) de su presentación visual (que se define en CSS). Esto permite que los autores de contenido puedan enfocarse en el contenido y los diseñadores puedan enfocarse en la forma en que se presenta ese contenido. CSS se utiliza para aplicar estilos a elementos como fuentes, colores, margenes y espaciados. También se puede utilizar para aplicar transformaciones y animaciones a elementos. CSS es una parte esencial de la creación de páginas web modernas y es compatible con la mayoría de los navegadores web.
